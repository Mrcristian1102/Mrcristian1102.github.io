CAMBIOS EN EL CSS
=================

1. VARIABLES DE COLOR Y TIPOGRAFÍA
- Se agregaron variables en :root para reutilizar colores
- Tamaños de fuente ahora con rem (responsive)
- Base: 16px en desktop, baja a 14px en móvil

2. RESPONSIVE DESIGN
Escritorio (1025px+):
- Menú horizontal
- Galería 3 columnas
- Proyectos 3 columnas

Tablets (768px - 1024px):
- Menú compacto
- Galería 2 columnas
- Proyectos 2 columnas

Móviles (hasta 768px):
- Menú en columna
- Galería 2 columnas
- Proyectos 1 columna

Móviles pequeños (hasta 480px):
- Todo en 1 columna
- Tipografía más pequeña

3. OPTIMIZACIÓN
- Reglas de CSS agrupadas para evitar repetición
- Selectores simples y claros
- Archivo minificado (style.min.css) para producción
- Todas las imágenes con max-width 100% y height auto

4. ARCHIVOS MODIFICADOS
- css/base.css - Variables base y tamaños de fuente
- css/estructura.css - Navegación, footer, elementos generales
- css/mascotas.css - Galería de mascotas responsive
- css/portfolio.css - Secciones del portafolio responsive
- css/style.min.css - Archivo minificado (nuevo)
